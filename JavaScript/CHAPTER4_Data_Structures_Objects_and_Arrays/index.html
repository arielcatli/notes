Chapter 4. Data Structures: Objects and Arrays

Objects allow us to group values—including other objects—together to build more complex structures.

Arrays

An array is a datatype specifically used for storing sequences of values. It is a list of values between square brackets.

let list_numbers = [2,3,4,5,2,3,545,34];
document.write(list_numbers[3]);

Properties

To access a property of a value, use the . operator.

            let list_numbers2 = [23,2,5,2,3,43,3,1];
            document.write(list_numbers2.length);

The Math object is a collection of math related constant and functions.

All objects in JavaScript have properties except the null and the undefined values.

Accessing Properties in JavaScript

  1. Dot 
                        let list = [2,3,34,1,3,3,4,5,3,];
                    document.write(list.length);
  2. Square Brackets 
                        let list2 = [23,34,5,23,34,54,24];
                    document.write(list2["length"]);

When using square brackets, the expression between the brackets is evaluated to get the property name. Whereas value.x fetches the property of value named “x”, value[x] tries to evaluate the expression x and uses the result as the property name.

Property names can have spaces in them. To reference it, use quotation marks.

The elements in an array are stored as the array’s properties, using numbers as property names.

Methods

Every string has the toUpperCase property which converts the string into uppercase.

Properties that contain functions are called methods.

Some Array Methods

  1. push 
                        let sequence = [1,2,3,23,3];
                    document.write(sequence);
                    sequence.push(99);
                    document.write(sequence);
  2. pop 
                        let sequence2 = [1,3,341,23,34,3];
                    document.write(sequence2);
                    document.write(sequence2.pop());
                    document.write(sequence2);
  3. shift (works with the first index - pop) 
                        let list3 = [2,3,5412,3];
                    document.write(list3);
                    list3.shift();
                    document.write(list3);
  4. unshift (works with the first index - push) 
                        let list4 = [12,3,32,31,23];
                    document.write(list4);
                    list4.unshift(99);
                    document.write(list4);
  5. indexOf

    This returns the index of the element in query. If it doesn't exists, the function will return -1. The search begins on the first index.

    It has a second argument which sets the start index of the search.

                        let list5 = [23,4,34,25,2];
                    document.write(list5);
                    document.write(list5.indexOf(23));
                    document.write(list5.indexOf(3));
  6. lastIndexOf

    This is similar to indexOf but it searches the last index first. The second argument sets the beginning of the search (from highest index to 0).

                        let list6 = [23,4,3,5,54,3,3,4];
                    document.write(list6);
                    document.write(list6.lastIndexOf(4));
                    document.write(list6.lastIndexOf(99));
  7. slice

    This returns a new array that is a subset of the sliced array. The first argument is the point of slicing (inclusive of the index) until the second argument (exclusive of this index). When the end index is not given, the slicing will continue to the end index of the sliced array.

                        let list7 = [23,3,5,23,23,5,23,5,5,3];
                    document.write(list7);
                    document.write(list7.slice(0,5));
  8. concat

    This function glues together two or more arrays and return a new array containing them. This is similar to the + operator when applied to arrays.

                        let glued = [];
                    glued = glued.concat([2,33,4,5,34,]);
                    document.write(glued);
                    glued = glued.concat([23,5,6,6,54,4]);
                    document.write(glued);
                    glued += [2,23,3,4,5,3,5,99];
                    document.write(glued);

    If you pass concat an argument that is not an array, that value will be added to the new array as if it was a one-element array.

When a property is added to immutable values, it is ignored by JavaScript.

Some String Methods

  • indexOf

    This is similar to searching for an element in an array. A substring with more than one character may be searched for.

  • trim

    This is used to trim whitespaces (spaces, newlines, tabs, and similar characters) from the start and the end of the string. 

                        document.write("     .hello.    ");
                    document.write("     .hello.    ".trim());
  • padStart

    This function pads a string with the character specified in the second argument. The first argument specifies the width of the string.

                        document.write(String("A").padStart(10, '*'));
  • split

    A string can be split with a delimiter using the split() method. It returns an array of the split strings. The first argument is the delimiter.

                        document.write("Hello world I am Ariel.".split(" "));
  • join

    To join together elements of an array or a string into a new string, use the join() method. The first index will be the delimiter for the new string returned.

                        document.write("Hello world I am Ariel".split(" ").join("."));
  • repeat

    To repeat a string, use the repeat() method. This returns a new string with the repeated substring. The first argument is the number of times to repeat the string.

                        document.write("HA".repeat(10));

Objects

Values of the type object are arbitrary collections of properties.

            let obj = 
            {
                name: "Ariel",
                age: 22,
                gender: "Male"
            };
            document.write(obj.name);

Reading a property that doesn't exist will return undefined.

A new value to a property can be assigned using the = operator. If the property doesn't exists, then the property will be created and assiged the value.

            document.write(obj.name);
            obj.name = "Mark";
            document.write(obj.name);
            document.write(obj.complexion);
            obj.complexion = "fair";
            document.write(obj.complexion);

The delete operator deletes the property of an object.

            document.write(obj.complexion);
            delete obj.complexion;
            document.write(obj.complexion);

The delete operator just severe the connection between the object and the 'deleted' value.

The in operator returns true if the value is in a string or a list value.

            let str = "Hello";
            document.write("name" in str);

The Object.assign() function copies all the properties to an object.

            Object.assign(listing, {name: "listing", purpose: "learning"});
            document.write(listing.name);
            document.write(listing.purpose);

Mutability

  • Immutable - values cannot be changed (numbers, strings, and Booleans).
  • Mutable - values can be changed (objects).
